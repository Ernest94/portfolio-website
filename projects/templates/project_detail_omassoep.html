Oma's Soep Data Analysis

About the project:

A data anlysis project. The weekly obtained Albert Heijn sales figures of Oma's Soep products are structured in an Excel file and data analysis is performed on these numbers. Besides the sales figures are saved in a SQL database and a data analysis Python Dashboard is created.


Technologies used:

Excel, Python, SQLite
